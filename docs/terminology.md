# Terminology

This document defines the canonical language of the Ship It! framework.

---

## Normative terms

These terms define the framework.

- Input
- Development
- Validation
- Ship

Do not replace these with synonyms in normative documents.

---

## Informative language

Marketing copy, website content and reflection statements may use different wording.

Example

Framework terminology:

Development

Website slogan:

Build.

Both are valid within their respective contexts.

---

## Principles

Prefer one canonical term over multiple similar terms.

Do not introduce new terms unless they define a genuinely new concept.
