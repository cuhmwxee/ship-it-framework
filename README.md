# Ship It!

The minimal software delivery framework.

> Build. Validate. Ship.

Ship It! is a lightweight software delivery framework focused on one thing:

Delivering working software safely.

## Core workflow

Input

↓

Development

↓

Validation

↓

Production

Validation that fails creates new input.

## Principles

- Keep it simple.
- Validate independently.
- Ship often.

## Status

🚧 Early development

Ship It! is actively evolving.

The framework is intentionally minimal.

Feedback and ideas are welcome.
